Include JUnit 5 + MockMvc tests, a `.github/copilot-instructions.md` with project conventions, and four `.github/agents/` profiles (api-dev, frontend-dev, code-reviewer, docs-writer). Run on port 3000, serve static files from `public/`, and make sure `./mvnw test` passes.
